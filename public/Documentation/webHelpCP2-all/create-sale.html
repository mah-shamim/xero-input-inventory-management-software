Create Sale

To create a sale, follow these steps:

Highlight important information.

  • Create Unit: Define the unit of measurement for your products.

  • Create Bank: Define the Bank.

  • Create Product Category: Categorize your products for better organization.

  • Create Brand: Specify the brand associated with your products.

  • Create Customer: Add the customers to whom you will sell products.

How to perform a task

Now you need to create a sale specifying product details.

How to create sale

  1. Select Product:

    • Navigate to the product search panel.

    • Select the desired product from the list.

  2. Input Sale Details:

    • Enter the quantity you want to sell using the selected unit of measurement.

    • Make necessary adjustments with price, discount, or adjustment.

    To be noted that, adjustment is not an adjusting figure, you can perform addition or subtraction.

  3. Payment Process:

    • Select the bank where you will receive the payment.

    • Complete the payment process.

  4. Finalize the Sale:

    • You can choose to print the sales receipt, view it, or simply submit it.
